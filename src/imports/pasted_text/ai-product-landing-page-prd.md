# PRD: AI Product Sprint Master Class 3기 랜딩 페이지

**작성일**: 2026-06-02
**작성자**: Super Human / 리사권
**버전**: v1.0
**상태**: Draft

---

## 1. Problem Statement

AI Product Sprint Master Class 3기 모집 기간 동안, 잠재 수강생이 프로그램 정보를 확인하고 신청 결정을 내릴 수 있는 전용 랜딩 페이지가 없다. 현재 정보가 여러 채널(DM, 오픈채팅, 인스타그램)에 분산되어 있어 구매 전환까지의 경로가 비효율적이다. 3기가 마지막 기수라는 희소성을 명확히 전달하지 못하면 관심 있는 잠재 수강생이 이탈할 가능성이 높다.

---

## 2. Goals

**유저 목표**
- G1. 방문자가 5분 안에 프로그램이 자신에게 맞는지 판단할 수 있다
- G2. 가격, 구성, 일정 등 구매 결정에 필요한 정보를 한 페이지에서 모두 확인한다
- G3. FAQ를 통해 구매 망설임의 주요 반론(가격, 시간, 수준)을 스스로 해소한다

**비즈니스 목표**
- G4. 페이지 방문자의 신청 전환율 목표: **5% 이상**
- G5. "3기가 마지막"이라는 희소성 메시지를 통해 결정 지연 방지
- G6. DM 문의 감소 — 반복 질문의 80%를 페이지에서 자기 해결 가능하도록

---

## 3. Non-Goals

- **결제 처리 기능 불포함**: 신청 버튼은 외부 결제 페이지(TossPayments / 아임웹)로 연결. 이 페이지에서 결제를 완료하지 않는다.
- **로그인/회원 기능 불포함**: 수강생 인증, 마이페이지, 강의 열람 기능은 범위 밖이다.
- **A/B 테스트 자동화 불포함**: 이번 버전은 단일 페이지. 다음 기수 이후 고려.
- **다국어 지원 불포함**: 한국어 단일 버전으로 출시. 영어 버전은 추후 검토.
- **CRM 연동 불포함**: 신청자 데이터 자동 수집은 아임웹 또는 별도 폼 도구에서 처리.

## 4. User Stories

### 페르소나 A — 관심은 있지만 망설이는 비개발자 직장인

- As a 직장인 잠재 수강생, I want 프로그램이 나 같은 코딩 비경험자에게 맞는지 빠르게 확인하고 싶다, so that 시간 낭비 없이 신청 여부를 결정할 수 있다.
- As a 직장인 잠재 수강생, I want 정해진 시간 참여가 필수인지 확인하고 싶다, so that 바쁜 일과 병행 가능한지 판단할 수 있다.
- As a 직장인 잠재 수강생, I want 실제 수강생 후기를 보고 싶다, so that 리사권 강사를 신뢰할 수 있는지 확인할 수 있다.

### 페르소나 B — 가격이 고민인 1인 창업가 / 프리랜서

- As a 가격에 민감한 잠재 수강생, I want 정가 대비 할인 구조를 명확히 보고 싶다, so that 지금 신청하는 게 최선인지 판단할 수 있다.
- As a 잠재 수강생, I want 3기가 정말 마지막인 이유를 납득할 수 있는 설명을 보고 싶다, so that 마감 압박이 진짜인지 가짜인지 확인하고 신뢰할 수 있다.
- As a 잠재 수강생, I want 환불 정책을 명확히 확인하고 싶다, so that 금전적 리스크 없이 신청 결정을 내릴 수 있다.

### 페르소나 C — 이미 신청 의향이 있는 방문자

- As a 신청 결정을 내린 방문자, I want 스티키 네비게이션의 CTA 버튼을 통해 언제든 신청 페이지로 이동할 수 있다, so that 스크롤 위치와 관계없이 즉시 행동할 수 있다.

## 5. Requirements

### P0 — Must Have (이 항목 없이는 출시 불가)

#### 5-1. 페이지 전체 구조 (9개 섹션)

| 섹션 | 내용 | 위치 |
|------|------|------|
| 히어로 | 뱃지 + 메인 카피 + 3개 숫자 지표 | 최상단 |
| 수강생 후기 | 실제 후기 3개 (카드 형식) | 2번째 |
| 공감/문제 제기 | 타겟 페인 포인트 리스트 + 인사이트 박스 | 3번째 |
| 강사 소개 | 크레덴셜 + 바이오 | 4번째 |
| 커리큘럼 | 토이 프로젝트 + 트랙 + 5주 로드맵 + 강의 수 | 5번째 |
| 3기 전용 구성 | 모각코 + 성과공유회 + 1:1 피드백 | 6번째 |
| 마지막 기수 이유 | 솔직한 브랜드 스토리 | 7번째 |
| 가격 + CTA | 가격표 + 앵커 박스 + 신청 버튼 | 8번째 |
| FAQ | 아코디언 7개 | 9번째 |

**인수 조건**
- [ ] 모든 섹션이 순서대로 렌더링된다
- [ ] 각 섹션에 `section-label`(소제목), `h2`(섹션 제목)가 있다
- [ ] 페이지 최하단에 최종 CTA 섹션이 있다

---

#### 5-2. 스티키 네비게이션

**인수 조건**
- [ ] 스크롤 내려도 상단에 고정된다
- [ ] 좌측: 브랜드명 `SUPER HUMAN`
- [ ] 우측: `3기 신청하기` 버튼 → `#pricing` 섹션으로 앵커 이동
- [ ] 배경: 반투명 블러 처리 (backdrop-filter)

---

#### 5-3. 가격 섹션

**가격 구조**

| 구분 | 가격 |
|------|------|
| 정가 | 590,000원 (취소선 표시) |
| 3기 마지막 할인 (15%) | 501,500원 |
| 3기 마지막 할인 + 리뷰 이벤트 (20%) | **472,000원** (강조, 별표) |

**인수 조건**
- [ ] 정가에 취소선(strikethrough) 적용
- [ ] 최저가 행은 배경색 또는 테두리로 강조
- [ ] 앵커 박스: "오프라인 부트캠프 300만원 vs 472,000원" 비교 문구 포함
- [ ] CTA 버튼: 흰색 배경, 검정 텍스트, `3기 신청하기`
- [ ] CTA 버튼 하단: "3기 이후 이 가격, 이 구조로는 열리지 않습니다" 노트

---

#### 5-4. FAQ 아코디언

**포함 질문 7개**
1. 코딩을 전혀 몰라도 되나요?
2. 가격이 좀 부담스러워요.
3. 나한테 맞는 수준인지 모르겠어요.
4. 정해진 시간이 있나요?
5. 사놓고 안 볼 것 같아서요.
6. 3기 이후에도 열리나요?
7. 환불이 가능한가요?

**인수 조건**
- [ ] 클릭 시 답변 열림/닫힘 토글
- [ ] 한 번에 하나만 열린다 (다른 항목 클릭 시 이전 항목 닫힘)
- [ ] `+` → `×` 아이콘 전환으로 상태 표시

---

#### 5-5. 비주얼 방향

블랙/화이트 모노톤으로 제작한다. 포인트 컬러 없음. 강조가 필요한 요소는 색상 대신 굵기(font-weight), 크기, 테두리, 배경 명도 차이로 처리한다.

**인수 조건**
- [ ] 배경: 검정 계열, 텍스트: 흰색 계열만 사용
- [ ] 포인트 컬러(노랑 등) 사용 금지
- [ ] 모든 섹션 하단에 구분선 적용
- [ ] 섹션 패딩: 80px 0 (모바일: 60px 0)

---

#### 5-6. 모바일 반응형

**인수 조건**
- [ ] 600px 이하에서 2열 그리드 → 1열로 전환 (히어로 지표, 강의 수, 맞는/안 맞는 분, 가격표)
- [ ] 5주 로드맵 테이블: 모바일에서 마지막 열 숨김
- [ ] 강사 소개 카드: 모바일에서 아바타-텍스트 세로 배치
- [ ] 모든 CTA 버튼: 모바일에서 전체 너비(100%)
